# Noble Forest RV Village — Visual Style Guide

*Last updated: June 2026 | Version 1.4*

---

## Emotional Thesis

Noble Forest RV Village must instantly evoke a profound sense of security, modern premium comfort, physical protection, and natural tranquility before a visitor reads a single sentence. The visual design directly contrasts the blinding, destructive glare of the open Texas heat with a cool, deeply covered, structured refuge. Every layout decision must signal an upscale, safe, and highly ordered residential environment for long-term guests.

---

## Color Token System

Configure these global color variables within the Elementor Site Settings panel:

* **Primary Brand Color (`--color-forest-green`):** `#1E4620` (Deep pine green used for header blocks, primary headings, layout banners, and hero accents).
* **Secondary/Accent Color (`--color-warm-earth`):** `#8D6E63` / `#A1887F` (Warm wood/earth tone used for primary call-to-action buttons, active hovers, card borders, and key features).
* **Background Light (`--color-cream-bg`):** `#FDFBF7` (Soft off-white canvas used behind dense information sections to reduce eye strain and establish a clean, premium editorial aesthetic).
* **Text Charcoal (`--color-charcoal`):** `#212121` (Deep charcoal body text color ensuring optimal contrast and readability compliance).

---

## Typography & Component Scale

* **Primary Heading Font System (H1, H2, H3):** *Montserrat* * *Treatment:* Bold (700) or Semi-Bold (600), solid tracking, clean geometric authority.
* **Body Font System:** *Inter* or *Open Sans*
    * *Treatment:* Regular (400), Medium (500), optimized line-height (1.6) to ensure dense policy lists and review texts remain scannable.

## Page Layout Specifics & Component Architecture

### The 3-Column Feature Matrix Components
On both the `/texas-renaissance-festival/` and `/rv-park-near-montgomery-texas/` landing templates, implement a clean, high-contrast, three-column icon/card grid detailing core selection metrics:
* **Row 1:** *Feel Safe* (Icon + Title + Bullet Points detailing gated infrastructure and onsite live-in manager features).
* **Row 2:** *Family Friendly* (Icon + Title + Bullet Points outlining community behavioral rules, rec room, and playground options).
* **Row 3:** *Great for Pets* (Icon + Title + Bullet Points detailing the 6,000+ sq. ft. dedicated shade park for pets to play in).

### Testimonial & Social Proof Carousels
* **Layout Design:** Clean vertical cards featuring a 5-star rating layout asset (`#FFD700` stars). Body copy uses Text Charcoal over an off-white container, highlighting real guest profiles (e.g., traveling contractors, long-term guests, festival attendees).

### 4-Column Footer Component Styling
* **Background Fill:** Full-width container using Primary Brand Color (`#1E4620`) or a rich muted slate variant.
* **Typography:** Titles use Montserrat Bold, White text. Body links and address metrics use clean off-white text with active Warm Earth hover transitions.
* **Layout Rule:** Column 1 features the high-contrast variant logo; Column 2 displays the scannable list of lower-case path targets; Column 3 presents social symbols and serving area targets; Column 4 anchors the clear text location data and responsive viewport map frame.

## What This Product Is Not

* **Not a Volatile, Fee-Heavy Corporate Chain:** Visually avoid multi-tiered billing matrix graphics or asterisks explaining utility surcharges. The layout must scream "one price, total clarity."
* **Not an Unregulated Campground:** Avoid chaotic text placements, script fonts, overlapping image grids, or messy borders. All components must follow strict Elementor grid and container layouts.
